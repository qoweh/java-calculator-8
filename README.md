## 1. 기능 목록
- 전체 로직 : 사용자는 문자열을 입력하고, 결과로서 구분자를 기준으로 분리된 숫자들의 합을 출력한다.
  - 부분 로직
    1. 초기 문자열("덧셈할 문자열을 입력해 주세요.\n")을 출력한다.
    2. 사용자에게 문자열을 입력받는다. [명시되지 않은 경우: 소수, 음수, 엄청 크거나 작은 수]
    3. 예외처리를 포함하여 문자열을 전처리한다.
    4. 결과를 특정 형태("결과 : 6\n")로 출력한다.
